Vjezba 14 - Preprocesori - Primjer 10

1. Kreirati html5 dokument osnovne strukture imena index.html

2. U dokumentu postaviti meta podatke
	- charset
	- viewport
	- http-equiv

3. Postaviti naslov dokumenta "SCSS"

4. U dokumentu kreirati slijedece elemente:

    <div class="flex-container">
        <div class="flex-item">1</div>
        <div class="flex-item">2</div>
        <div class="flex-item">3</div>
        <div class="flex-item">4</div>
    </div>

5. Kreirati datoteku style.scss sa slijedećim pravilima

// import
@import "mixins";

$color1: #eee;
$color2: #333;

* {
  box-sizing: border-box;
}

div {
  background-color: $color1;
  border: 2px solid $color2;
  text-align: center;
  padding: 1em;
}

.flex-container {
  @include flex;
  min-height: 200px;
}
.flex-item {
  flex-basis: 23%;
  @include mq("large", max) {
    flex-basis: 48%;
    margin-bottom: 1em;
  }
  @include mq("small", max) {
    flex-basis: 100%;
  }
}

6. Kreirati datoteku _mixins.scss sa slijedećim pravilima

@mixin flex {
  display: flex;
  flex-wrap: wrap;
  justify-content: space-between;
}

//media queries

$breakpoints: (
  "small": 480px,
  "medium": 768px,
  "large": 1024px,
  "x-large": 1200px
);

//keywords($args) //=> map
@mixin mq($width, $type: min) {
  // map-has-key($map, $key)
  @if map_has_key($breakpoints, $width) {
    //map-get($map, $key)
    $width: map_get($breakpoints, $width);
    @if $type == max {
      $width: $width - 1px;
    }
    @media (#{$type}-width: $width) {
      @content;
    }
  }
}
