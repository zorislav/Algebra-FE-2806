6. Kreirati html5 dokument osnovne strukture imena heading-font-size.html

7. U dokumentu postaviti meta podatke
	- charset
	- viewport
	- http-equiv

8. Postaviti naslov dokumenta "Svojstva za uređivanje fontova"

9. Pomocu internog CSS-a definirati slijedeća pravila

        h1 
            font-size: 2em;
            /* 2 * 16px = 32px */

        h2 
            font-size: 1.5em;
            /* 1.5 * 16px = 24px */

        h3 
            font-size: 1.17em;
            /* 1.17 * 16px = 18.72px */

        h4 
            font-size: 1em;
            /* 1 * 16px = 16px */

        h5 
            font-size: .83em;
            /* 0.83 * 16px = 13.28px */

        h6 
            font-size: .67em;
            /* 0.67 * 16px = 10.72px */
